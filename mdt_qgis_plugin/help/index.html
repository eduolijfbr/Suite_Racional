Manual do MDT Generator

O MDT Generator é uma ferramenta técnica para transformar curvas de nível em superfícies contínuas (rasters).

1. Preparação dos Dados

Certifique-se de que sua camada de curvas de nível:

  • Está em um Sistema de Coordenadas Projetadas (ex: UTM / SIRGAS 2000), não em coordenadas geográficas (Graus).
  • Possui um campo de atributo numérico com a elevação (cota).

2. Parâmetros

  • Layer Select: Camada vetorial de origem.
  • Z/Elevation Field: O campo que será usado como a altura para cada linha.
  • Resolution: O tamanho do pixel no mapa. Recomenda-se usar o dobro da equidistância das curvas (ex: curvas de 1m -> resolução 2m) para um equilíbrio entre performance e detalhe.

3. O Algoritmo

O plugin utiliza uma técnica de Interpolação Linear (TIN). As curvas são densificadas com pontos a cada 1 metro para evitar que o modelo perca precisão em áreas de alta curvatura.

Dica Geográfica: O plugin alinha o "canto" do raster a múltiplos arredondados da resolução. Isso garante que, se você gerar dois rasters com a mesma resolução em projetos diferentes, os pixels estarão perfeitamente sobrepostos.

4. Resolução de Problemas

Se o processo falhar:

  1. Verifique se o disco tem espaço suficiente (rasters de 0.1m podem ocupar centenas de megabytes).
  2. Certifique-se de que não há letras especiais ou espaços excessivos no nome do arquivo de saída.
  3. Confira o painel de log em Mensagens Log > MDT Plugin no QGIS.
