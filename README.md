Suíte Racional
A Suíte Racional é um conjunto de plugins para o QGIS desenvolvido para simplificar e acelerar o fluxo de trabalho de profissionais de engenharia e geoprocessamento no dimensionamento de sistemas de microdrenagem urbana.

O projeto centraliza o processo de cálculo hidrológico e dimensionamento hidráulico em um único ambiente SIG, eliminando a necessidade de alternar entre planilhas externas e softwares de mapeamento.

🛠️ Componentes da Suíte
A suíte é composta pelos seguintes módulos:

Método Racional Pro: Plugin principal para o dimensionamento de galerias de microdrenagem.
Gerador de MDT: Ferramenta para criação de Modelos Digitais de Terreno (MDT) a partir de curvas de nível do projeto.
Medir 3D: Ferramenta complementar para medição e análise de parâmetros altimétricos diretamente no ambiente SIG.
🚀 Principais Funcionalidades
Baseado no fluxo de trabalho do Método Racional Pro:

Integração de Dados Espaciais: Extração automática de parâmetros altimétricos a partir de MDT.
Análise de Impermeabilidade: Classificação visual sobre imagens de satélite para definição ágil do coeficiente de escoamento (C).
Hidrologia Regionalizada: Cálculo do Tempo de Concentração (Tc) por diversos métodos (Kirpich, Giandotti, Ventura) e módulo IDF configurável para dados locais.
Dimensionamento Automático: Cálculo de galerias com verificações de velocidade, lâmina d'água e regime de escoamento (Número de Froude).
Perfil Longitudinal: Geração instantânea de perfis longitudinais da rede de drenagem e lançamento rápido de Poços de Visita (PVs).
Memória de Cálculo: Geração de relatórios padronizados com todos os parâmetros e memórias de cálculo para aprovação técnica.
📋 Requisitos e Limitações
Software: Desenvolvido exclusivamente para o ambiente QGIS.
Aplicação: Recomendado para bacias de microdrenagem de até 50-100 hectares. Para áreas maiores ou análises dinâmicas complexas, recomenda-se o uso de ferramentas como o SWMM.
Este projeto visa oferecer transparência técnica e eficiência na gestão de infraestrutura urbana.
