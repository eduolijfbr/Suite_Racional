1. Introdução

O Método Racional Pro é um plugin para QGIS que automatiza o cálculo de drenagem urbana utilizando o Método Racional. Desenvolvido para engenheiros, projetistas e estudantes, o plugin oferece:

  • ✅ Extração automática de parâmetros do MDT (Modelo Digital de Terreno)
  • ✅ Cálculo de vazão pelo Método Racional
  • ✅ Dimensionamento de condutos pela equação de Manning
  • ✅ Curvas IDF de diversas cidades brasileiras
  • ✅ Verificações técnicas automáticas
  • ✅ Geração de relatórios profissionais
  • ✅ Exportação em múltiplos formatos
💡 O que é o Método Racional?
O Método Racional é uma metodologia amplamente utilizada para estimar a vazão de pico em pequenas bacias hidrográficas urbanas. É recomendado para áreas de até 2 km² e considera a intensidade da chuva, a área da bacia e o coeficiente de escoamento superficial.

2. Instalação

2.1 Requisitos

  • QGIS versão 3.4 ou superior
  • Sistema operacional: Windows, Linux ou macOS

2.2 Passos para Instalação

1

Localizar a pasta de plugins

No QGIS, vá em Configurações → Perfis de Usuário → Abrir Pasta do Perfil Ativo, depois navegue até python/plugins/

Windows: %APPDATA%\QGIS\QGIS3\profiles\default\python\plugins\

2

Copiar o plugin

Copie a pasta metodo_racional_pro inteira para a pasta de plugins.

3

Reiniciar o QGIS

Feche completamente o QGIS e abra novamente.

4

Ativar o plugin

Vá em Complementos → Gerenciar e Instalar Complementos, procure por "Método Racional Pro" e marque a caixa para ativar.

✅ Pronto! O plugin estará disponível em Complementos → Método Racional Pro

3. Interface do Plugin

3.1 Acessando o Plugin

Após a instalação, acesse o plugin por:

  • Menu: Complementos → Método Racional Pro → Método Racional - Cálculo de Drenagem
  • Barra de ferramentas: Clique no ícone na barra "MetodoRacionalPro"

3.2 Visão Geral da Interface

A interface é dividida em duas áreas principais:

Área Descrição
Painel Esquerdo Camadas do projeto, dados de entrada e precipitação. No topo, o botão 📘 GUIA TÉCNICO oferece acesso rápido às limitações normativas.
Painel Direito Resultados, verificações técnicas e opções de exportação.

4. Camadas do Projeto

O plugin utiliza camadas carregadas no projeto QGIS para extrair automaticamente os parâmetros da bacia.

4.1 Tipos de Camadas

Camada Tipo Uso
MDT Raster Cálculo de desnível (altitude máx - mín)
Área de Estudo Polígono Delimitação da bacia - cálculo de área
Uso do Solo Polígono Cálculo do coeficiente de impermeabilidade
Curvas de Nível Linha Alternativa ao MDT para desnível
Cursos d'água Linha Cálculo da distância do talvegue
💡 Dica: Selecione feições específicas no mapa antes de clicar em "EXTRAIR PARÂMETROS". Se nenhuma feição estiver selecionada, o plugin usará todas as feições da camada.
⚠️ Nota de Limitação Técnica: O Método Racional é estritamente recomendado para bacias de microdrenagem e áreas urbanas pequenas (geralmente < 50-100 hectares). Para bacias de macrodrenagem, os resultados devem ser validados via Hidrograma Unitário (SCS) ou SWMM.

4.2 Extração Automática

Clique no botão "🔄 EXTRAIR PARÂMETROS" para calcular automaticamente:

  • Área: Calculada a partir do polígono selecionado
  • Desnível: Diferença entre altitude máxima e mínima do MDT na área
  • Distância: Comprimento do curso d'água ou maior reta inscrita no polígono
  • Impermeabilidade: Coeficiente ponderado pelo uso do solo
  • Tempo: Calculado pelo método de Kirpich

5. Dados de Entrada

5.1 Parâmetros

DISTÂNCIA (m)

Comprimento do talvegue principal da bacia

DESNÍVEL (m)

Diferença de altitude entre montante e jusante

TEMPO (min)

Tempo de concentração da bacia

IMPERMEABILIDADE

Coeficiente de escoamento (0 a 1)

ÁREA (km²)

Área total da bacia hidrográfica

RUGOSIDADE

Coeficiente de Manning do conduto

DECLIVIDADE (%)

Declividade longitudinal do conduto

5.2 Coeficientes de Rugosidade (Manning)

Material n (típico) Faixa
PEAD 0.010 0.009 - 0.012
PVC 0.010 0.009 - 0.011
Concreto 0.013 0.012 - 0.015
Galeria Celular 0.015 0.013 - 0.017

5.3 Coeficientes de Escoamento (C)

Tipo de Superfície C (típico)
Área urbana impermeável (asfalto, concreto) 0.90 - 0.95
Área comercial/industrial 0.80 - 0.90
Área residencial alta densidade 0.60 - 0.75
Área residencial baixa densidade 0.40 - 0.55

Floresta/mata 0.10 - 0.20

6. Realizando Cálculos

6.1 Passo a Passo

1

Carregar camadas no QGIS

Carregue o MDT, polígono da área de estudo e opcionalmente uso do solo e cursos d'água.

2

Selecionar a área de interesse

No mapa, selecione o polígono da bacia que deseja calcular (opcional).

3

Abrir o plugin

Vá em Complementos → Método Racional Pro → Método Racional - Cálculo de Drenagem

4

Selecionar as camadas

Nos combos de "Camadas do Projeto", selecione as camadas correspondentes.

5

Extrair parâmetros

Clique em "🔄 EXTRAIR PARÂMETROS" para calcular automaticamente os dados.

6

Ajustar valores

Verifique e ajuste os valores de rugosidade, declividade e outros conforme necessário.

7

Calcular intensidade

Clique em "📊 Calcular IDF" para obter a intensidade de precipitação.

8

Dimensionamento e Verificação

Clique em "CALCULAR". Verifique o Número de Froude e a Velocidade para garantir a estabilidade do conduto conforme o Guia Técnico.

7. Curvas IDF

As curvas IDF (Intensidade-Duração-Frequência) relacionam a intensidade da chuva com sua duração e tempo de retorno.

7.1 Equação IDF

I = (K × TRa) / (t + b)c
I = Intensidade (mm/h) | TR = Tempo de retorno (anos) | t = Duração (min)

7.2 Cidades Disponíveis

O plugin inclui curvas IDF para as seguintes cidades brasileiras:

Brasília - DF

Salvador - BA

7.3 Curvas Personalizadas

Você pode adicionar curvas IDF personalizadas:

  1. Clique em "📊 Calcular IDF"
  2. Clique em "➕ Nova"
  3. Informe o nome da localidade
  4. Marque "Habilitar edição dos parâmetros"
  5. Preencha os valores de K, a, b, c
  6. Clique em "💾 Salvar Parâmetros"
💾 Persistência: As curvas personalizadas são salvas automaticamente e estarão disponíveis nas próximas sessões.

8. Resultados e Verificações

8.1 Resultados do Cálculo

Resultado Descrição Unidade
Vazão Vazão de projeto calculada pelo Método Racional m³/s
Diâmetro Diâmetro calculado pela equação de Manning m
Galeria Diâmetro comercial recomendado m
Velocidade Velocidade de escoamento no conduto m/s

8.2 Verificações Técnicas

✓ Velocidade Adequada
Limites: 0,6 m/s (mínimo para autolimpeza) a 5,0 m/s (máximo para evitar erosão)
⚠️ Escoamento e Regime (Froude)
  • Regime Subcrítico (Fr < 0.8): Escoamento estável e tranquilo.
  • Regime Crítico/Instável (0.8 ≤ Fr ≤ 1.2): Zona de instabilidade na lâmina d'água. Deve ser evitada.
  • Regime Supercrítico (Fr > 1.2): Escoamento torrencial. Requer atenção à dissipação de energia.
✓ Lâmina d'água
Relação lâmina/diâmetro (y/D) ideal entre 50% e 85%.
⚠ Atenção: Se alguma verificação apresentar alerta, revise os parâmetros de projeto ou considere soluções alternativas.

10. Fórmulas Utilizadas

10.1 Método Racional

Q = (C × I × A) / 3,6
Q = Vazão (m³/s) | C = Coef. escoamento | I = Intensidade (mm/h) | A = Área (km²)

10.2 Tempo de Concentração (Kirpich)

Tc = 57 × (L³ / H)0,385
Tc = Tempo de concentração (min) | L = Comprimento (km) | H = Desnível (m)

10.3 Equação de Manning

Q = (1/n) × A × R2/3 × S1/2
Q = Vazão (m³/s) | n = Rugosidade | A = Área seção (m²) | R = Raio hidráulico (m) | S = Declividade (m/m)

10.4 Número de Froude

Fr = V / √(g × y)
Fr = Número de Froude | V = Velocidade (m/s) | g = 9,81 m/s² | y = Profundidade hidráulica (m)
